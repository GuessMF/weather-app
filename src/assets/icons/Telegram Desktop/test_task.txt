Тестовое задание:

Задача - написать приложение на Next JS или React JS, которое позволяет пользователям показывать погоду для выбранного города, используя открытые API, как пример - openweather

Стек:
- React JS или Next JS
- Redux

Функциональные требования:
- Возможность выбрать город из выпадающего списка или ввести его вручную
- Должно использоваться открытое API для получения погоды
- Возможность сохранять город, чтобы пользователю при входе не нужно заново вводить данные
- Возможность удалять и добавлять различные города для отслеживания избранных, с возможностью их последующего просмотра
- Присутствие базовой анимации
- Предоставление пользователю возможности настройки единиц измерения (например, градусы Цельсия или Фаренгейта)

(Приветствуется изобретательность и дополнение функционала)

Нефункциональные требования:
- Адаптивность под различные размеры экранов
- Обеспечение безопасности данных пользователей при работе с API и хранении информации о выбранных городах
- Реализация кроссбраузерной совместимости для работы в различных веб-браузерах
- Документирование кода для лучшей поддерживаемости и расширяемости
- Применение лучших практик разработки в соответствии с принципами SOLID

Будет плюсом показать навыки тестирования отдельных компонентов системы
